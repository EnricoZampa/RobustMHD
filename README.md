# RobustMHD
Hcurl based pressure, fluid Reynold and magnetic Reynolds robust finite element method.

Author: Enrico Zampa, University of Vienna 

The code was developed with NGSolve 6.2.2504.

Numerical experiments can be run from command line, e.g. 
python3 MHD_solver.py --ictype 1 --NMAX 20 --order 2 --nu 1e-3 --eta 1e-4 --dt 0.1
